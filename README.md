# Hospitality Performance Dashboard
This repository showcases a hospitality performance dashboard developed using Microsoft Excel to analyze and track overall business performance through structured data analysis and visualization. The project involves cleaning and organizing raw data, creating calculated columns and measures, and designing meaningful KPIs to represent key performance indicators. The dashboard includes interactive slicers, filters, and dynamic charts that allow users to explore trends, compare performance across different categories, and derive actionable insights. The objective of this project is to present complex data in a clear, visually intuitive, and business-friendly format that supports effective decision-making and performance evaluation.
